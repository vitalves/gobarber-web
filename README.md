# GoBarber

## Configurações iniciais feitas com:**

`yarn add eslint -D`
`yarn eslint --init`

`yarn add prettier eslint-config-prettier eslint-plugin-prettier babel-eslint -D`

`yarn add eslint-plugin-react-hooks -D`

## Estrutura de rotas:

`yarn add react-router-dom`

## Navegação entre telas:

`yarn add history`
